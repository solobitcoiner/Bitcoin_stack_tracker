Bitcoin Stack Tracker is a clean, dark-themed web app that shows you the current value of your Bitcoin holdings and projects how much they could be worth over time.
You enter your BTC amount, and it automatically pulls the live Bitcoin price. A growth rate slider (default 20%) lets you model different scenarios, and you can project across 5, 10, 20, or 30 year horizons. 
The results are shown as a summary, a live chart, and a milestone table breaking down key years.
It's a simple, no-login, single-file tool — just open it in a browser and it works.
